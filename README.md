## Sobre o projeto
Este projeto mostra como usar IndexedDB para armazenamento de dados no lado do cliente com Blazor Auto e C#.

IndexedDB é uma API de banco de dados nativa para navegadores que permite o armazenamento de grandes quantidades de dados. Um banco de dados NoSQL projetado para armazenar dados estruturados, incluindo arquivos e blobs. É ideal para aplicações que precisam funcionar offline, como PWA (Progressive Web Apps). Ele pode armazenar dados localmente para sincronizar com o servidor quando a conexão for restaurada. Pode ser usado como um cache de dados para reduzir as solicitações de rede e melhorar o desempenho da aplicação.

O Blazor é uma estrutura Web de front-end do .NET para criar UI interativas usando C# e .NET. Ele oferece diferentes modos de renderização, <code>None</code>, <code>Server</code>, <code>WebAssembly</code> e o <code>Auto</code>. O modo de renderização <code>Auto</code> é um recurso que permite ao Blazor escolher automaticamente o modo mais adequado de renderização (servidor ou cliente) com base no contexto. Isso otimiza o desempenho e a experiência do usuário, evitando sobrecarga desnecessária.
